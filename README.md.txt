# 🎨 Color Detection from Image

This app lets users upload an image, click on it, and detect the closest named color based on RGB values using OpenCV and Streamlit.

## 🚀 Features

- 📁 Upload an image
- 🖱️ Click anywhere to detect a color
- 🎨 View RGB and HEX values
- 🏷️ See the closest matching color name
- 🖼️ Visual swatch of detected color
